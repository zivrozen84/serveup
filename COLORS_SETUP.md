# הגדרת צבעי התפריט

המדריך מסביר איך להפעיל ולהשתמש ב־5 צבעים מותאמים בתפריט המסעדה.

---

## שלב 1: סנכרון מסד הנתונים

ודא שה־schema במסד הנתונים כולל את שדות הצבע. הרץ בטרמינל:

```bash
cd c:\Users\Ziv\Desktop\Serveup
npx prisma db push
```

זה מעדכן את טבלת `restaurants` עם העמודות:
- `secondaryColor` – צבע כותרות קטגוריות
- `textColor` – צבע שמות מנות
- `descriptionColor` – צבע תיאור
- `priceColor` – צבע מחיר

---

## שלב 2: יצירת Prisma Client (אם נדרש)

אם מופיעה שגיאה על Prisma Client, הרץ:

```bash
npx prisma generate
```

---

## שלב 3: הפעלת האפליקציה

```bash
npm run dev
```

---

## שימוש

1. היכנס לפאנל האדמין.
2. בחר מסעדה לעריכה.
3. בחלק **"צבעים"** תמצא 5 בוחרים בשורה:
   - **צבע ראשי** – רקע כותרת, כפתורי קטגוריות
   - **צבע משני** – כותרות קטגוריות (למשל "מנות ראשונות")
   - **צבע טקסט** – שמות המנות
   - **צבע תיאור** – טקסט התיאור
   - **צבע מחיר** – המחיר
4. לחץ **"עדכן"** כדי לשמור.

## פתרון בעיות

**"Expected string, received null"**  
- בדוק שה־schema ב־`lib/validation/restaurant.ts` משתמש ב־`.nullish()` לשדות הצבע.

**הצבעים לא משתנים בתפריט**  
- ודא ש־`npx prisma db push` הור בהצלחה.
- ודא ששמרת בטופס (כפתור "עדכן").
- ריענן את דף התפריט.

**אין עמודות חדשות ב-DB**  
- הרץ שוב: `npx prisma db push`
- ודא ש־PostgreSQL פועל ו־`.env` מכיל `DATABASE_URL` תקין.
